PodPulse

A containerized web application deployed using Docker and Kubernetes. This project demonstrates deployment, service exposure, replica scaling, and pod-based application hosting.

Dockerized Kubernetes Deployment Scalable Nginx Hosted

Project Highlights

  • Built as a static web application and packaged inside a Docker container.
  • Deployed on Kubernetes using Deployment and Service manifests.
  • Supports replica scaling and pod self-healing behavior.
  • Served using Nginx inside a lightweight Alpine-based container.

Deployment Status

The application is currently running inside a Kubernetes pod.
